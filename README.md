## ğŸ¶ About Strudel

[Strudel](https://strudel.cc/) is a JavaScript-based live coding environment inspired by TidalCycles. It runs in the browser and allows expressive, pattern-based music creation.

This repository is meant to:
- Archive finished songs
- Store works-in-progress
- Reuse samples across compositions
